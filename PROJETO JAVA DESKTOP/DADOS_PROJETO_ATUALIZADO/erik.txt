Sistema deve ser um chat.
- Sistema deve ser executável
- Deve permitir login
- Deve gerenciar contatos
- Sistema deve contatar o servidor deve fazer o meio de campo no processo de login
- Sistema deve exibir contatos adicionados
- Sistema deve exibir contatos online e offline
- Sistema tem que ter uma tela de contato com outra pessoa online
- Sistema deve identificar quem esta online ou offline (por imagem, por grupo, cores);
- Sistema deve ter logout.
- Sistema deve ser automatizado
- Sistema deve permitir a aceitaçãod e um contato automaticamente
- Sistema deve avisar SUCESSO ou INVÁLIDO ao cadastrar um contato
- Sistema deve avisao SUCESSO ou INVÁLIDO ao fazer login
- Sistema deve emitr aviso de contato duplicado adicionado
- Sistema deve emitr um som e piscar ao alguem falar 
- Sistema deve abrir a tela

RN - Sistema so pode entrar no chat se tiver o login e senha

RN6 - Mensagens geradas pelo servidor para um cliente devem ser enviadas atraves da porta 56001 - CLIENTE NAO PODE RESPONDER ESSAS MENSAGENS.
